# Part 107 Prep — iOS/macOS App

## Opening in Xcode

The SwiftUI source files are here in `apps/ios/Part107Prep/`. To build and run:

### 1. Create the Xcode Project

1. Open **Xcode** → **File** → **New** → **Project**
2. Choose **Multiplatform** → **App**
3. Product Name: `Part107Prep`
4. Organization Identifier: `com.yourname.part107prep`
5. Interface: **SwiftUI**
6. Language: **Swift**
7. Save it into the `apps/ios/` directory

### 2. Add Source Files

Drag the following into the Xcode project:
- `Part107Prep/Models/Question.swift`
- `Part107Prep/ViewModels/QuizViewModel.swift`
- `Part107Prep/Part107PrepApp.swift`

### 3. Add Question Data

Copy the JSON files from `packages/content/questions/` into the Xcode project as **resources**:
- `regulations.json`
- `airspace.json`
- `weather.json`
- `operations.json`
- `loading_performance.json`

Make sure "Add to Target: Part107Prep" is checked.

### 4. Configure Targets

The project should automatically support:
- **iPhone** (iOS 17+)
- **iPad** (iPadOS 17+)
- **Mac** (macOS 14+ via Mac Catalyst or native macOS target)

### 5. Build & Run

- Select your target device (iPhone simulator, iPad, or My Mac)
- Press **⌘R** to build and run

## Architecture

```
Part107Prep/
├── Models/
│   └── Question.swift          # Data models (matches JSON schema)
├── ViewModels/
│   └── QuizViewModel.swift     # Quiz engine (study + exam modes)
├── Views/                      # (build out as you go)
│   ├── HomeView.swift
│   ├── StudyView.swift
│   ├── ExamView.swift
│   └── ChartViewer.swift
└── Part107PrepApp.swift        # App entry point + navigation
```

The SwiftUI app shares the same question bank JSON files as the web app.
Any new questions generated by the content pipeline are immediately available to both platforms.
